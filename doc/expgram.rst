=======
expgram
=======

--------------------------
query ngram language model
--------------------------

:Author: Taro Watanabe <taro.watanabe@nict.go.jp>
:Date:   2013-2-8
:Manual section: 1

SYNOPSIS
--------

**expgram** [*options*]

DESCRIPTION
-----------



OPTIONS
-------

  --ngram arg   ngram in ARPA or expgram format
  --input arg   input
  --output arg  output
  --order arg   ngram order
  --shard arg   # of shards (or # of threads)
  --verbose     verbose level
  --debug       debug level
  --help        help message

ENVIRONMENT
-----------

TMPDIR_SPEC
  temporary directory
