expgram_diff
============

--------------------------------------------

--------------------------------------------

:Author: Taro Watanabe <taro.watanabe@nict.go.jp>
:Date:   2013-3-14
:Manual section: 1

SYNOPSIS
--------

**expgram_diff** [*options*] *ngram(s)*

DESCRIPTION
-----------



OPTIONS
-------

  **--output** `arg (="-")`    output result

  **--temporary** `arg`        temporary directory

  **--shard** `arg (=4)`       # of shards (or # of threads)

  **--debug** `[=arg(=1)]`     debug level

  **--help** help message

ENVIRONMENT
-----------

TMPDIR_SPEC
  temporary directory
